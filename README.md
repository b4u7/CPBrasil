# CPBrasil - Desktop Application
A desktop application that allows you to play CPBrasil.

## Installation
To install the application head over to the Releases tab and download the latest asset.

## Development

### Getting Started
Before working on the application it is recommended to fully read this document
and to refer back to and edit throughout development.

Start development by downloading or cloning this repository.

### Linting your code
To check if there are any issues with your code run the `npm run lint` command.
To fix those issues run `npm run lint-fix`.

### Recommendations
When committing your code it is recommended to follow [conventional commits](https://www.conventionalcommits.org/en/v1.0.0/), however it is not mandatory.
Make sure to explain your changes thoroughly when creating a pull-request.
